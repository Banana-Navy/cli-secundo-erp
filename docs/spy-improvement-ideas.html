2. 20 idees d'amelioration

Tier S Impact eleve, implementation rapide

1 Tableau de bord comparatif multi-concurrents
Vue tableau comparant TOUS les concurrents cote a cote : temps de reponse, nombre de followers, note Google, CMS utilise. Calcul automatique : "Vous etes N-ieme sur X concurrents" par metrique. 100% front-end, les donnees existent deja dans les snapshots.
2 Historique et graphiques de tendances
Exploiter les snapshots horodates pour tracer l'evolution : followers Instagram, temps de reponse, note Google. Sparklines sur chaque carte, graphiques detailles dans la vue detail. Lib : Recharts (gratuit, React). Les donnees historiques sont DEJA stockees.
3 Systeme d'alertes automatiques
Alertes sur seuils configurables : "Concurrent X a depasse 10 000 followers", "Le temps de reponse de Y a double". Table spy_alerts dans Supabase + notification via src/lib/notifications.ts deja present.
4 Score de menace concurrent (Threat Score)
Algorithme composite : croissance followers, note Google, temps de reponse, frequence de publication, taille equipe LinkedIn. Score 0-100 avec code couleur (vert/orange/rouge). Tri des concurrents par score. Calcul 100% front-end.
5 Scan automatique programme (Cron)
Utiliser pg_cron de Supabase + Edge Functions pour scan automatique quotidien/hebdomadaire. Limite gratuite : 500 000 invocations Edge Functions/mois, pg_cron inclus.
6 Analyse du sitemap.xml et robots.txt
Fetcher /sitemap.xml et /robots.txt : nombre total de pages, structure du contenu, pages bloquees au crawl, frequence de MAJ (<lastmod>). Extension simple du scraper existant.

Tier A Impact eleve, faisabilite moyenne

7 Analyse de performance web via Google APIs gratuites
PageSpeed Insights API : 25 000 req/jour — mesurer LCP, CLS, INP. CrUX API : metriques reelles d'utilisateurs Chrome, 150 req/sec. Nouveau type de snapshot : "performance".
8 Detection de changements de contenu
Hash SHA-256 du <title> + meta description + headings H1/H2. Si hash different : alerte "Le concurrent X a modifie sa page d'accueil". Extension : stocker le diff textuel. API Wayback Machine gratuite pour historique supplementaire.
9 Analyse des mots-cles SEO a partir du HTML
Extraire H1, H2, H3, title, meta description, alt-text des images. Nuage de mots-cles par concurrent. Comparer entre concurrents pour identifier les opportunites manquees. Regex sur HTML deja fetche, cout = 0.
10 Rapport automatique PDF/email hebdomadaire
Resume genere par Gemini API (1 000 req/jour gratuit) : "Cette semaine, le concurrent X a gagne 200 followers, Y a ameliore son temps de 15%". Email via Supabase Edge Function.
11 Veille des annonces immobilieres concurrentes
Scraper les pages de listing : compter le nombre de biens affiches, extraire les fourchettes de prix via JSON-LD RealEstateListing/Offer. Suivi : "Le concurrent X affiche 45 biens (+5 cette semaine)".
12 Analyse WHOIS et expiration de domaine
Whoxy : 100% gratuit, illimite. WhoisJSON : 1 000 req/mois. Who-Dat : open source, self-hostable. Informations : date creation, expiration, registrar, nameservers.

Tier B Impact moyen, faisabilite variable

13 Detection de la stack technique
Au-dela du generator : detecter WordPress (wp-content/), GA (gtag), Facebook Pixel (fbq), Hotjar, HubSpot via patterns HTML/JS. CRFT Lookup et Stackcrawler : gratuits et illimites.
14 Verification SSL et headers de securite
Mozilla Observatory : gratuit, illimite. Analyser HSTS, CSP, X-Frame-Options. Scoring securite exploitable en marketing. Headers HTTP accessibles via fetch() natif.
15 Suivi des publications et frequence de contenu
Detecter les flux RSS (/feed/, /rss.xml, <link rel="alternate">). Compter la frequence de publication : "Le concurrent X publie 3 articles/semaine".
16 Analyse de l'empreinte carbone web
Website Carbon API : gratuite. Comparer l'eco-responsabilite. Argument marketing : "Notre site est X% plus ecologique que nos concurrents".
17 Benchmark des donnees de marche
FRED : prix immobilier historiques Espagne. INE : indices officiels espagnols. Spanish Property Insight DataHub : donnees notaires. KPIs marche a cote des donnees concurrentielles.
18 Matrice SWOT automatique generee par IA
Agglomerer toutes les donnees d'un concurrent, envoyer a Gemini API gratuit pour generer une analyse SWOT en francais. Nouvel onglet "Analyse IA" dans la fiche concurrent.
19 Carte geographique des concurrents
Adresses depuis JSON-LD (deja parse). Leaflet.js : gratuit, pas de cle API. Google Maps Embed API : 100% gratuit. Identifier les zones de forte concentration concurrentielle.
20 Monitoring de la reputation en temps reel
Scraper Google pour "nom concurrent" avis / "nom concurrent" arnaque". Analyse de sentiment via Gemini. Source additionnelle : pages publiques Trustpilot.

4. Techniques d'extraction zero-cout

Techniques utilisant uniquement fetch() natif + regex, deja dans la stack :

1. Parser sitemap.xml + robots.txt
Compter les URLs, extraire les <lastmod>, categoriser les pages (blog, biens, destinations)
2. Detecter les outils marketing
Google Analytics (gtag/ga), Facebook Pixel (fbq), Hotjar, HubSpot, Mailchimp via regex JS
3. Extraire les headers HTTP
server, x-powered-by, content-security-policy, strict-transport-security
4. Detecter les langues
Chercher <link rel="alternate" hreflang="X"> — important pour l'immobilier international
5. Auto-decouvrir les flux RSS
<link rel="alternate" type="application/rss+xml">
6. Compter les images + alt-text
Estimer la richesse visuelle du site, extraire les alt pour analyse SEO
7. Detecter les chatbots
Intercom, Drift, Tawk.to, WhatsApp widget via patterns HTML/JS
8. Analyser les liens externes
Identifier les partenaires, portails immobiliers, affilies du concurrent
